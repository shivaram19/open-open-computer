## P5-5: End-to-End Autonomous Task

**What:** Give harness a real task with no human intervention.

**Example:** "Build a voice agent for Telugu-speaking parents in Suryapet that can handle school fee payment queries."

**Success criteria:**
- Harness decomposes task autonomously
- Twins research and debate without human input
- Consensus reached on architecture
- Code generated and tested
- Deployed and monitored
- Self-evaluation report produced
